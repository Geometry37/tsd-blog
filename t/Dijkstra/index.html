Dijkstra

条件：所有边权均大于 0

松弛操作：

优点：算法中用贪心，效率较高

缺点：只能用于正权边图

# 步骤

  1. 将所有顶点分成两部分：已求出最短路径的顶点和未求出最短路径的顶点 *（vis 数组）*
  2. dis 数组初始化： dis[s] = 0, dis[i] = (i ≠ s)
  3. 在 Q 中选一个 dis 值最小的顶点 u 加入 P 中，并考虑 u 的所有出边，对每一条进行松弛
  4. 重复执行第 3 步，一直到 Q 中无顶点结束
