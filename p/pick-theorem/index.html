# Pick 定理 [笔记][计算几何]

# 基本概念

给定顶点均为整点的简单多边形，该定理说明了其面积 AAA 和内部个点数目 iii、边上格点数目 bbb 的关系：

A=i+b2−1A=i+\frac{b}{2}-1 A=i+2b​−1

详细的证明可以看【这里】，在本文简述一下，反正不打算投日报

# 简略证明

因为所有简单多边形都可切割为一个三角形和另一个简单多边形。考虑一个简单多边形 PPP，及跟 PPP 有一条共同边的三角形 TTT。若 PPP 符合 Pick 公式，则只要证明 PPP 加上 TTT 的 PTPTPT 亦符合皮克公式，以及三角形符合皮克公式，就可根据数学归纳法，对于所有简单多边形皮克公式都是成立的。
